# 보석 도둑

**출처**: 백준 1202
**유형**: 그리디 / 우선순위 큐
**제한**: 시간 2초 | 메모리 256MB | N, K ≤ 300,000

## 문제

N개의 보석이 있고 각 보석은 무게 Mᵢ와 가치 Vᵢ를 가진다. K개의 배낭이 있고 각 배낭에는 보석을 최대 1개만 넣을 수 있으며, 배낭마다 최대 담을 수 있는 무게 Cᵢ가 다르다. 훔칠 수 있는 보석 가치의 합의 최댓값을 구하라.

## 입력

첫째 줄에 N과 K. 다음 N개의 줄에 보석의 무게와 가치. 다음 K개의 줄에 배낭의 최대 무게.

## 출력

훔칠 수 있는 보석 가치 합의 최댓값.

## 예제 1

**입력**
```
3 2
1 65
5 23
4 40
10
2
```
**출력**
```
105
```

## 예제 2

**입력**
```
2 1
5 10
3 20
6
```
**출력**
```
20
```
