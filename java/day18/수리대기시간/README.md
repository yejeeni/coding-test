# 수리 대기 시간

**출처**: 백준 스타일
**유형**: 그리디 / 정렬
**제한**: 시간 1초 | 메모리 256MB | N ≤ 100,000

## 문제

N명의 고객이 줄을 서서 컴퓨터 수리를 기다리고 있습니다. 수리 기사는 한 번에 한 명만 수리할 수 있고, 각 고객의 수리 시간은 t_i분입니다. 수리가 끝날 때까지 나머지 고객은 계속 기다려야 합니다. 모든 고객의 대기 시간(자신의 수리가 끝나는 시각)의 합을 최소화하세요.

## 입력

첫째 줄에 N, 둘째 줄에 N개의 수리 시간이 주어진다.

## 출력

대기 시간의 합의 최솟값을 출력한다.

## 예제 1

**입력**
```
3
3 1 2
```
**출력**
```
10
```

## 예제 2

**입력**
```
4
1 2 5 6
```
**출력**
```
26
```
