# 회의실 배정

**출처**: 백준 스타일
**유형**: 그리디 / 정렬
**제한**: 시간 1초 | 메모리 256MB | N ≤ 100,000

## 문제

N개의 회의가 있고 각 회의는 시작 시간과 끝 시간이 주어집니다. 회의실이 1개일 때, 겹치지 않게 최대한 많은 회의를 배정하려고 합니다. 단, 한 회의가 끝나는 동시에 다른 회의를 시작할 수 있습니다.

## 입력

첫째 줄에 N이 주어진다. 둘째 줄부터 N개의 줄에 각 회의의 시작 시간과 끝 시간이 주어진다.

## 출력

배정할 수 있는 최대 회의 수를 출력한다.

## 예제 1

**입력**
```
5
1 4
3 5
0 6
5 7
3 8
```
**출력**
```
2
```

## 예제 2

**입력**
```
3
1 2
2 3
1 3
```
**출력**
```
2
```
