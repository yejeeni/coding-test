# 숨바꼭질

**출처**: 백준 1697
**유형**: BFS / 1차원 탐색
**제한**: 시간 2초 | 메모리 128MB | N, K ≤ 100,000

## 문제

수빈이는 위치 N에 있고, 동생은 위치 K에 있다. 수빈이는 1초에 다음 세 가지 중 하나를 선택할 수 있다.

- 현재 위치 X에서 **X-1** 로 이동
- 현재 위치 X에서 **X+1** 로 이동
- 현재 위치 X에서 **2*X** 로 이동

수빈이가 동생을 찾는 **최소 시간(초)** 을 구하라.

## 입력

첫째 줄에 N과 K가 주어진다.

## 출력

최소 시간을 출력한다.

## 예제 1

**입력**
```
5 17
```
**출력**
```
4
```

## 예제 2

**입력**
```
5 5
```
**출력**
```
0
```
