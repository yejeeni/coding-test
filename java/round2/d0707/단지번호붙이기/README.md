# 단지번호붙이기

**출처**: 백준 2667
**유형**: BFS / 격자 탐색
**제한**: 시간 1초 | 메모리 128MB | N ≤ 25

## 문제

0과 1로 이루어진 N×N 정사각형 지도가 있다. `1`은 집이 있는 칸, `0`은 없는 칸이다. 상하좌우로 연결된 집들의 집합을 **단지**라고 한다.

단지의 수와, 각 단지에 속하는 집의 수를 오름차순으로 출력하라.

## 입력

첫째 줄에 N이 주어진다.
다음 N줄에 N개의 숫자가 **붙어서** 주어진다.

## 출력

첫째 줄에 단지 수, 이후 각 단지의 집 수를 오름차순으로 한 줄씩 출력.

## 예제 1

**입력**
```
7
0110100
0110101
1110101
0000111
0100000
0111110
0111000
```
**출력**
```
3
7
8
9
```
