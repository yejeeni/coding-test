# N과 M (1)

**출처**: 백준 15649
**유형**: 백트래킹 / 순열
**제한**: 시간 1초 | 메모리 512MB | 1 ≤ M ≤ N ≤ 8

## 문제

1부터 N까지 자연수 중 M개를 골라 순서 있게 나열한 수열을 **사전순**으로 모두 출력하라. 같은 수를 여러 번 고를 수 없다.

## 입력

```
N M
```

## 출력

한 줄에 하나씩 수열 출력

## 예제 1

**입력**
```
3 2
```
**출력**
```
1 2
1 3
2 1
2 3
3 1
3 2
```

## 예제 2

**입력**
```
4 1
```
**출력**
```
1
2
3
4
```
