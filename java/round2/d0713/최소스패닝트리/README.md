# 최소 스패닝 트리

**유형**: MST / 크루스칼
**제한**: 시간 1초 | 메모리 128MB | V ≤ 10,000 / E ≤ 100,000

## 문제

그래프가 주어졌을 때, 최소 스패닝 트리의 가중치 합을 구하라.

## 입력

첫째 줄에 V(정점 수), E(간선 수)
다음 E줄에 `A B C` (A-B를 잇는 가중치 C인 간선)

## 출력

최소 스패닝 트리의 가중치 합

## 예제 1

**입력**
```
3 3
1 2 1
2 3 2
1 3 3
```
**출력**
```
3
```

## 예제 2

**입력**
```
4 4
1 2 5
1 3 3
2 4 2
3 4 1
```
**출력**
```
6
```
