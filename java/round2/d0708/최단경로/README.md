# 최단경로

**유형**: 다익스트라
**제한**: 시간 1초 | 메모리 256MB | V ≤ 20,000 / E ≤ 300,000

## 문제

방향 가중치 그래프가 주어진다. 시작 정점 K에서 다른 모든 정점까지의 최단 경로를 구하라.

## 입력

첫째 줄에 정점 수 V, 간선 수 E
둘째 줄에 시작 정점 K
이후 E줄에 `u v w` (u→v, 비용 w)

## 출력

V개 줄에 걸쳐 1번 정점부터 순서대로 최단 거리 출력. 도달 불가능하면 `INF`

## 예제 1

**입력**
```
5 6
1
5 1 1
1 2 2
1 3 3
2 3 4
2 4 5
3 4 6
```
**출력**
```
0
2
3
7
INF
```
