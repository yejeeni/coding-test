# 용돈 관리

## 문제
현우는 앞으로 N일 동안 정확히 M번만 통장에서 돈을 인출해서 쓰려 한다.
매번 K원을 인출하며, 하루 사용 금액이 남은 돈보다 많으면 남은 돈을 넣고 다시 K원을 인출한다.
정확히 M번을 맞추기 위해, 남은 돈이 충분해도 강제로 반납하고 다시 인출할 수 있다.
인출 금액 K의 최솟값을 구하라.

## 제한
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ N
- 하루 사용 금액: 1 이상 10,000 이하

## 입력 형식
```
첫째 줄: N M
둘째 줄~: N개의 줄에 i번째 날 사용 금액
```

## 출력 형식
```
최소 인출 금액 K 출력
```

## 예제 입력 1
```
7 5
100
400
300
100
500
101
400
```

## 예제 출력 1
```
500
```
