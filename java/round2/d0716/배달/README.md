# 배달 비용 최소화

**유형**: 그리디 / 정렬
**제한**: 시간 1초 | 메모리 256MB | N ≤ 100,000

## 문제

택배 기사가 N개의 물건을 배달해야 한다. 각 물건에는 무게 $w_i$가 있고, 트럭은 한 번에 최대 $C$kg까지 실을 수 있다.

물건은 한 번에 여러 개 실을 수 있지만, **무게 합이 C를 초과할 수 없다**. 트럭이 창고를 나갔다 돌아오는 횟수를 **최소화**하려면 몇 번 왕복해야 하는가?

단, 각 물건의 무게는 $w_i ≤ C$ 이므로 혼자서는 반드시 실을 수 있다.

## 입력

첫째 줄에 N과 C가 주어진다.  
둘째 줄에 N개의 물건 무게 $w_1, w_2, ..., w_N$이 주어진다.

## 출력

최소 왕복 횟수를 출력한다.

## 예제 1

**입력**
```
6 10
1 9 3 7 2 8
```
**출력**
```
3
```

## 예제 2

**입력**
```
4 10
5 5 5 5
```
**출력**
```
2
```
