# 격자 탈출

**유형**: 구현 / 방향 시뮬레이션
**제한**: 시간 1초 | 메모리 256MB | N, M ≤ 100

## 문제

N×M 크기의 격자가 있다. 각 칸에는 방향(U/D/L/R) 중 하나가 적혀 있다.

로봇이 시작 위치 (r, c) (0-indexed)에서 출발한다. 로봇은 현재 칸에 적힌 방향으로 한 칸 이동하며, 격자 밖으로 나가면 이동을 멈추고 탈출한다.

이동 중 이미 방문한 칸에 다시 도달하면 무한 루프로 보고 -1을 출력한다.

탈출할 때까지 이동한 횟수를 출력하라.

## 입력

첫 번째 줄에 N M이 주어진다.
다음 N줄에 M개의 방향 문자(U/D/L/R)가 공백으로 구분되어 주어진다.
마지막 줄에 시작 위치 r c가 주어진다. (0-indexed)

## 출력

탈출 횟수 또는 -1을 출력한다.

## 예제 1

**입력**
```
3 4
R D L U
U R D L
L U R D
0 0
```
**출력**
```
6
```

## 예제 2

**입력**
```
2 2
R L
D U
0 0
```
**출력**
```
-1
```
