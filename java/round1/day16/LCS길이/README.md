제목: LCS 길이

시간 제한: 1초 | 메모리: 256MB | 1 ≤ 문자열 길이 ≤ 1,000

문제:

두 문자열 A, B가 주어질 때
LCS(최장 공통 부분 수열)의 길이를 출력하라.

부분 수열은 문자열에서 일부 문자를 순서를 유지한 채 골라낸 것이다.

예: "ACGT"의 부분 수열 → "ACT", "GT", "A" 등


입력 형식:

첫째 줄에 문자열 A
둘째 줄에 문자열 B

출력 형식:

LCS의 길이를 출력한다.

예제 입력 1:
```
ACAYKP
CAPCAK
```

예제 출력 1:
```
4
```

예제 입력 2:
```
ABCD
EFGH
```

예제 출력 2:
```
0
```
