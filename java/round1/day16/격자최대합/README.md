제목: 격자 최대합

시간 제한: 1초 | 메모리: 256MB | 1 ≤ N, M ≤ 1,000

문제:

N×M 격자가 주어진다. (1,1)에서 (N,M)까지
오른쪽 또는 아래로만 이동할 수 있다.
지나는 칸의 값의 합이 최대가 되도록 이동할 때, 그 최댓값을 출력하라.

입력 형식:

첫째 줄에 N, M이 주어진다.
다음 N줄에 M개의 정수가 주어진다. (0 ≤ 값 ≤ 100)

출력 형식:

최댓값을 출력한다.

예제 입력 1:
```
3 3
1 2 3
4 5 6
7 8 9
```

예제 출력 1:
```
29
```

예제 입력 2:
```
2 2
0 0
0 0
```

예제 출력 2:
```
0
```
