## 친구 관계 전이

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N명의 학생이 있다. Q개의 쿼리가 주어질 때,
아래 두 종류의 쿼리를 처리하여라.

- `1 a b`: a와 b를 친구로 연결한다.
- `2 a b`: a와 b가 같은 친구 그룹이면 YES, 아니면 NO를 출력한다.

### 제한
- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000

### 입력 형식
첫째 줄에 N, Q가 주어진다.
다음 Q개의 줄에 쿼리가 주어진다.

### 출력 형식
타입 2 쿼리마다 YES 또는 NO를 출력한다.

### 예제 입력 1
```
5 5
1 1 2
1 2 3
2 1 3
1 4 5
2 1 4
```

### 예제 출력 1
```
YES
NO
```
