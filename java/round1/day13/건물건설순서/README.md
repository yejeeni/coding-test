## Q3. 건물 건설 순서 (도전)

**유형**: 위상정렬 + 누적 시간
**난이도**: 골드 3
**출처 스타일**: 백준

**문제**
N개의 건물이 있고, 각 건물 건설에는 일정 시간이 걸린다.
일부 건물은 다른 건물이 완공된 후에만 지을 수 있다.
각 건물의 완공 시간을 구하라. (병렬 건설 가능)

**제한**
- 시간: 2초 / 메모리: 256MB
- 1 ≤ N ≤ 1,000
- 0 ≤ M ≤ 100,000
- 건설 시간: 1 ≤ t ≤ 100,000

입력 형식
- 첫 줄에 N, M
- 둘째 줄에 각 건물의 건설 시간 t[1] ~ t[N]
- 이후 M줄에 a b (a가 완공되어야 b 착공 가능)

출력 형식
- 각 건물의 최소 완공 시간을 N줄에 출력

예제 입력
```
5 4
1 2 3 2 4
1 3
2 3
3 5
4 5
```

예제 출력
```
1
2
5
2
9
```
