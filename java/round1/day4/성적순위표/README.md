### 🟡 문제 2 — 성적 순위표 (메인)

**[프로그래머스 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 이름은 알파벳 소문자, 길이 ≤ 10
- 점수는 1 ≤ score ≤ 100,000
- 이름은 중복 없음

**문제**
N명의 이름과 점수가 주어진다. 다음 두 가지 쿼리를 처리하라.

- `rank name` — name의 등수를 출력 (동점자가 있으면 같은 등수, 등수는 나보다 점수 높은 사람 수 + 1)
- `top K` — 점수 상위 K명의 이름을 점수 내림차순으로 출력 (동점이면 이름 오름차순)

**입력 형식**
```
첫째 줄: N
다음 N줄: name score
다음 줄: 쿼리 수 Q
다음 Q줄: 쿼리
```

**출력 형식**
```
각 쿼리 결과를 한 줄씩
top K 쿼리는 이름을 공백으로 구분해 한 줄 출력
```

**예제 입력 1**
```
4
alice 90
bob 85
carol 90
dave 70
1
rank alice
```
**예제 출력 1**
```
1
```

**예제 입력 2**
```
4
alice 90
bob 85
carol 90
dave 70
2
top 3
rank bob
```
**예제 출력 2**
```
alice carol bob
3
```

> 💡 체크포인트: `HashMap`으로 이름→점수 저장, `ArrayList`+`Comparator`로 정렬. 동점 처리에서 다중 정렬 키 (`b[1]!=a[1]`이면 점수 비교, 같으면 이름 비교) 를 람다로 써봐요.
