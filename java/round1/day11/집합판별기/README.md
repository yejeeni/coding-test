## 문제 1. 집합 판별기

시간 제한: 1초 | 메모리 제한: 256MB | N ≤ 1,000,000

1부터 N까지의 원소가 있다.
M개의 연산이 주어진다.

연산의 종류:
- 0 a b : a와 b를 같은 집합으로 합친다
- 1 a b : a와 b가 같은 집합이면 "YES", 아니면 "NO" 출력

입력 형식
첫째 줄에 N, M이 주어진다.
이후 M줄에 걸쳐 연산이 주어진다.

출력 형식
1 연산에 대해 YES 또는 NO를 출력한다.

예제 입력
```
5 6
0 1 2
0 3 4
1 1 2
1 1 3
0 2 3
1 1 4
```

예제 출력
```
YES
NO
YES
```
