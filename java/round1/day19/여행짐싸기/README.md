# 여행 짐 싸기

**유형**: 0/1 Knapsack
**제한**: 시간 1초 | 메모리 256MB | N ≤ 100, W ≤ 1,000

## 문제

여행을 떠나는 준규는 N개의 짐 후보가 있다. 각 짐은 무게 wᵢ와 필요도 vᵢ를 가진다. 배낭의 최대 무게는 W이다. 배낭에 넣을 수 있는 짐의 필요도 합의 최댓값을 구하라. 각 짐은 한 번씩만 넣을 수 있다.

## 입력

첫째 줄에 N과 W. 다음 N개의 줄에 무게와 필요도.

## 출력

필요도 합의 최댓값.

## 예제 1

**입력**
```
4 5
2 3
3 4
1 2
4 5
```
**출력**
```
7
```

## 예제 2

**입력**
```
3 3
2 2
3 5
1 3
```
**출력**
```
5
```
