## Q3. 경유지를 거치는 최단 경로 (도전)

**유형**: 다익스트라 2회 응용  

**문제**  
N개의 노드, 양방향 간선으로 이루어진 그래프가 있다.  
출발지 S에서 반드시 경유지 V1, V2를 **모두 거쳐** 목적지 E까지 가는 최단 경로를 구하라.  
경유 순서는 상관없다 (V1→V2 또는 V2→V1 중 짧은 것).

**제한**
- 시간: 1초 / 메모리: 128MB
- 1 ≤ N ≤ 800
- 1 ≤ M ≤ 200,000
- 가중치: 1 ≤ w ≤ 100
- 경로가 없으면 -1 출력

입력 형식  
첫 줄에 N, M  
이후 M줄에 a b c (양방향, 가중치 c)  
마지막 줄에 S V1 V2 E

출력 형식  
최단 경로 길이. 불가능하면 -1

예제 입력
```
4 4
1 2 1
2 3 2
3 4 1
2 4 10
1 2 3 4
```

예제 출력
```
4
```
