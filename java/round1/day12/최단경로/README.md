## Q1. 최단 경로 (워밍업)

**유형**: 다익스트라 기본  

**문제**  
방향 그래프가 주어진다. 1번 노드에서 다른 모든 노드까지의 최단 거리를 구하라.  
도달할 수 없는 노드는 INF를 출력한다.

**제한**
- 시간: 1초 / 메모리: 256MB
- 노드 수 V: 1 ≤ V ≤ 20,000
- 간선 수 E: 1 ≤ E ≤ 300,000
- 가중치: 1 ≤ w ≤ 10

입력 형식  
첫 줄에 V, E  
둘째 줄에 시작 노드 번호 K (= 1)  
이후 E줄에 u v w (u→v, 가중치 w)

출력 형식  
V줄. i번 노드까지 최단 거리, 도달 불가면 INF

예제 입력
```
5 6
1
1 2 2
1 3 3
2 3 1
2 4 1
3 4 4
4 5 2
```

예제 출력
```
0
2
3
3
5
```
