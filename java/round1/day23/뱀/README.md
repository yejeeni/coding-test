# Q3. 뱀

**유형**: 구현 / 시뮬레이션 / 덱  
**제한**: 시간 1초 | 메모리 128MB | N ≤ 100

## 문제
N×N 격자에 뱀이 있다. 뱀은 (1,1)에서 시작하고 처음에는 오른쪽을 향한다. 매 초마다 다음을 수행한다:

1. 머리를 다음 칸으로 이동
2. 이동한 칸에 사과가 있으면 사과를 먹고 꼬리 유지 (몸이 길어짐)
3. 사과가 없으면 꼬리 제거
4. 이동한 칸이 벽이거나 자기 몸이면 게임 종료

방향 전환 명령이 주어진다. **X초가 끝난 뒤** D(오른쪽 회전) 또는 L(왼쪽 회전).

게임이 끝나는 시간(초)을 출력하라.

## 입력
```
N
K (사과 수)
K줄: 사과 위치 r c
L (방향 전환 수)
L줄: X 방향
```

## 출력
게임이 끝나는 시간(초)

## 예제 1
**입력**
```
6
3
3 4
2 5
5 3
3
3 D
15 L
17 D
```
**출력**
```
9
```
