# 백준 12865 — 평범한 배낭

**출처**: 백준 12865
**유형**: 0/1 Knapsack
**제한**: 시간 2초 | 메모리 512MB | N ≤ 100 / K ≤ 100,000

## 문제

N개의 물건이 있다. 각 물건은 무게 W와 가치 V를 가진다.
최대 무게 K까지 담을 수 있는 배낭에 물건을 담을 때,
가치의 합이 최대가 되도록 하라. 각 물건은 최대 1번만 담을 수 있다.

## 입력

첫째 줄에 N, K가 주어진다.
다음 N줄에 W, V가 주어진다.

## 출력

최대 가치를 출력한다.

## 예제 1

**입력**
```
4 7
6 13
4 8
3 6
5 12
```
**출력**
```
14
```
