# 도시 연결

**유형**: MST / 크루스칼
**제한**: 시간 1초 | 메모리 256MB | N ≤ 10,000 / M ≤ 100,000

## 문제

N개의 도시와 M개의 도로가 있다. 각 도로는 두 도시를 연결하며 비용이 있다. 모든 도시를 연결하는 최소 비용을 출력하라. 연결이 불가능하면 -1을 출력하라.

## 입력

첫째 줄에 N, M이 주어진다.
다음 M줄에 a, b, c가 주어진다. (도시 a와 b를 비용 c로 연결)

## 출력

최소 비용 또는 -1

## 예제 1

**입력**
```
4 5
1 2 3
1 3 1
2 3 2
2 4 4
3 4 5
```
**출력**
```
7
```

## 예제 2

**입력**
```
3 1
1 2 5
```
**출력**
```
-1
```
