### 🔴 문제 3 — 좌표 압축 (도전)

**[백준 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- -1,000,000,000 ≤ 각 수 ≤ 1,000,000,000

**문제**
N개의 수가 주어진다. 각 수를 0부터 시작하는 순위(같은 수는 같은 순위)로 변환해서 출력하라.

**입력 형식**
```
첫째 줄: N
둘째 줄: N개의 정수
```

**출력 형식**
```
변환된 순위를 공백으로 구분해 한 줄 출력
```

**예제 입력 1**
```
5
2 4 -10 4 -9
```

**예제 출력 1**
```
2 3 0 3 1
```

**예제 입력 2**
```
4
1000000000 -1000000000 0 0
```

**예제 출력 2**
```
2 0 1 1
```
