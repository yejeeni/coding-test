### 🟡 문제 2 — 회의실 배정 (메인)

**[백준 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 0 ≤ 시작시간 < 끝시간 ≤ 1,000,000

**문제**
N개의 회의가 주어진다. 각 회의는 시작시간과 끝시간이 있다.
회의실이 하나뿐일 때, 겹치지 않게 최대한 많은 회의를 배정하라.
끝나는 시간과 시작하는 시간이 같으면 연속으로 사용 가능하다.

**입력 형식**
```
첫째 줄: N
다음 N줄: 시작시간 끝시간
```

**출력 형식**
```
최대 회의 수
```

**예제 입력 1**
```
5
1 4
3 5
0 6
5 7
3 8
```

**예제 출력 1**
```
2
```

**예제 입력 2**
```
3
1 2
2 3
3 4
```

**예제 출력 2**
```
3
```
