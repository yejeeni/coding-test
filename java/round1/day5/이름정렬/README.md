### 🟢 문제 1 — 이름 정렬 (워밍업)

**[백준 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 이름 길이 ≤ 20, 알파벳 소문자

**문제**
N개의 이름이 주어진다. 다음 기준으로 정렬해서 출력하라.
1. 이름 길이 오름차순
2. 길이가 같으면 알파벳 오름차순

**입력 형식**
```
첫째 줄: N
다음 N줄: 이름 하나씩
```

**출력 형식**
```
정렬된 이름을 한 줄에 하나씩
```

**예제 입력 1**
```
5
banana
apple
fig
kiwi
plum
```

**예제 출력 1**
```
fig
kiwi
plum
apple
banana
```

💡 체크포인트: 길이 비교는 a.length() - b.length(), 알파벳 비교는 a.compareTo(b).
