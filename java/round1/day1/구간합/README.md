## 🚀 문제 2 (메인) — 구간 합
`[백준 스타일]`

첫 줄에 정수 N과 Q가 주어진다.
둘째 줄에 N개의 정수가 공백으로 주어진다.
다음 Q줄에는 두 정수 l, r이 주어진다. 각 줄마다 l번째부터 r번째 수(1-indexed)의 합을 출력하시오.

**📌 제한**
*   **시간:** 1초 / **메모리:** 256MB
*   1 ≤ N, Q ≤ 100,000
*   각 정수: 0 ≤ x ≤ 10^9
*   1 ≤ l ≤ r ≤ N

**예제 입력 1**
```text
5 3
1 2 3 4 5
1 3
2 5
1 5
```

**예제 출력 1**
```text
6
14
15
```

**예제 입력 2**
```text
3 2
1000000000 1000000000 1000000000
1 3
2 3
```

**예제 출력 2**
```text
3000000000
2000000000
```

> **⚠️ 힌트 아닌 주의:** 예제 2의 출력이 `int` 범위를 넘는지 확인해 보세요! (`long` 타입 사용을 고려해야 합니다.)
