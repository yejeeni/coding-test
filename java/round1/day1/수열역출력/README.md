## 📝 문제 1 (워밍업) — 수열 역출력
`[백준 스타일]`

첫 줄에 정수 N이 주어진다.
둘째 줄에 N개의 정수가 공백으로 주어진다.
이 수들을 역순으로 공백으로 구분하여 한 줄에 출력하시오.

**📌 제한**
*   **시간:** 1초 / **메모리:** 256MB
*   1 ≤ N ≤ 100,000
*   각 정수: -10^9 ≤ x ≤ 10^9

**예제 입력 1**
```text
5
3 1 4 1 5
```

**예제 출력 1**
```text
5 1 4 1 3
```

**예제 입력 2**
```text
3
-7 0 2000000000
```

**예제 출력 2**
```text
2000000000 0 -7
```
