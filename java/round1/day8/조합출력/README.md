# [워밍업] 조합 출력

## 문제
1부터 N까지의 수 중에서 M개를 골라 오름차순으로 출력하라.
같은 수를 여러 번 고를 수 없고, 고른 수는 오름차순이어야 한다.

## 제한
- 시간: 1초 / 메모리: 256MB
- 1 ≤ M ≤ N ≤ 8

## 입력 형식
```
첫째 줄: N M
```

## 출력 형식
```
조합을 한 줄에 하나씩 출력 (오름차순)
```

## 예제 입력 1
```
4 2
```

## 예제 출력 1
```
1 2
1 3
1 4
2 3
2 4
3 4
```

## 예제 입력 2
```
3 1
```

## 예제 출력 2
```
1
2
3
```
