# [메인] 팀 나누기

## 문제
N명(N은 짝수)을 N/2명씩 두 팀으로 나눌 때,
두 팀의 능력치 합 차이의 최솟값을 구하라.

## 제한
- 시간: 1초 / 메모리: 256MB
- 2 ≤ N ≤ 20 (N은 짝수)
- 1 ≤ 능력치 ≤ 100

## 입력 형식
```
첫째 줄: N
둘째 줄: N명의 능력치 (공백 구분)
```

## 출력 형식
```
두 팀 능력치 합 차이의 최솟값
```

## 예제 입력 1
```
4
1 2 3 4
```

## 예제 출력 1
```
0
```

## 예제 입력 2
```
2
10 20
```

## 예제 출력 2
```
10
```
