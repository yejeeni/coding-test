# [도전] 랜선 자르기

## 문제
K개의 랜선이 있다. 각 랜선을 잘라서 길이가 모두 같은 랜선 N개를 만들고 싶다.
남는 부분은 버린다. 만들 수 있는 랜선의 최대 길이(정수)를 구하라.

## 제한
- 시간: 2초 / 메모리: 256MB
- 1 ≤ K ≤ 10,000
- 1 ≤ N ≤ 1,000,000
- 각 랜선 길이: 1 이상 2^31-1 이하
- 항상 N개를 만들 수 있는 입력만 주어진다

## 입력 형식
```
첫째 줄: K N
다음 K줄: 랜선 길이
```

## 출력 형식
```
최대 길이 출력
```

## 예제 입력 1
```
4 11
802
743
457
539
```

## 예제 출력 1
```
200
```

## 예제 입력 2
```
2 5
10
10
```

## 예제 출력 2
```
3
```
