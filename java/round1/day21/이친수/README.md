# 이친수 (백준 2193)

**유형**: 1D DP
**제한**: 시간 2초 | 메모리 128MB | N ≤ 90

## 문제

0과 1로만 이루어진 수를 이진수라 한다. 이 중 아래 조건을 만족하는 수를 **이친수**라 한다.

1. 이친수는 0으로 시작하지 않는다.
2. 이친수에서 1이 두 번 연속으로 나타나지 않는다.

N자리 이친수의 개수를 구하라.

## 입력
N이 주어진다.

## 출력
N자리 이친수의 개수를 출력한다.

## 예제 1

**입력**
```
3
```
**출력**
```
2
```

## 예제 2

**입력**
```
4
```
**출력**
```
3
```
