# 추가 2. 크로아티아 알파벳 (백준 2941)

**출처**: 백준 2941
**유형**: 문자열 / replace
**제한**: 시간 1초 | 메모리 128MB

## 문제

크로아티아어에는 아래 특수 알파벳이 있다. 이 알파벳들은 **두 글자 이상**으로 표현된다.

`c=` `c-` `dz=` `d-` `lj` `nj` `s=` `z=`

문자열이 주어졌을 때, 몇 개의 크로아티아 알파벳으로 이루어져 있는지 출력하라.

## 입력
한 줄에 문자열 (길이 ≤ 100)

## 출력
알파벳 수.

## 예제 1
**입력**
```
ljes=njak
```
**출력**
```
6
```

## 예제 2
**입력**
```
ddz=z=
```
**출력**
```
3
```
