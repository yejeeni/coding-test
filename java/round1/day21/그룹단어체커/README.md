# 추가 1. 그룹 단어 체커 (백준 1316)

**출처**: 백준 1316
**유형**: 문자열 / 문자 처리
**제한**: 시간 2초 | 메모리 128MB | N ≤ 100

## 문제

그룹 단어란 단어에 존재하는 모든 문자에 대해, 각 문자가 **연속해서만** 나타나는 단어다.

예) `aabbbccdd` → 그룹 단어 ✓  
예) `aabba` → 그룹 단어 ✗ (`a`가 `b` 이후 다시 등장)

N개의 단어 중 그룹 단어의 개수를 출력하라.

## 입력
첫째 줄에 N, 이후 N줄에 단어.

## 출력
그룹 단어의 개수.

## 예제 1
**입력**
```
3
happy
new
year
```
**출력**
```
3
```

## 예제 2
**입력**
```
1
aabba
```
**출력**
```
0
```
