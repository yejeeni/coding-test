# 로봇 청소기

**출처**: 백준 14503
**유형**: 구현 / 시뮬레이션
**제한**: 시간 2초 | 메모리 512MB | N, M ≤ 50

## 문제

N×M 크기의 방에 로봇 청소기가 있다. 방은 빈 칸과 벽으로 구성되어 있고, 로봇은 아래 규칙으로 동작한다.

1. 현재 칸이 청소되지 않은 경우 청소한다.
2. 주변 4칸 중 청소되지 않은 빈 칸이 없는 경우:
    - 바라보는 방향 유지한 채로 한 칸 후진 → 1번으로
    - 후진도 불가능하면 작동 멈춤
3. 주변 4칸 중 청소되지 않은 빈 칸이 있는 경우:
    - 반시계 방향으로 90° 회전
    - 앞쪽 칸이 청소되지 않은 빈 칸이면 전진
    - 1번으로

청소한 칸의 수를 출력하라.

## 입력

첫째 줄: N M  
둘째 줄: r c d (시작 위치, 방향 0=북 1=동 2=남 3=서)  
이후 N줄: 방 상태 (0=빈칸, 1=벽)

## 출력

청소한 칸의 수

## 예제 1

**입력**
```
3 3
1 1 0
1 1 1
1 0 1
1 1 1
```
**출력**
```
1
```
