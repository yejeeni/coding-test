# 작업 스케줄러

**유형**: 그리디 / 우선순위 큐
**제한**: 시간 1초 | 메모리 256MB | N ≤ 100,000

## 문제

N개의 작업이 있다. 각 작업은 **마감 시한 D**와 **보상 W**가 있다. 하루에 한 작업만 처리할 수 있고, 마감 시한 D일 **이내**에 처리하면 보상을 받는다. 마감 시한을 넘긴 작업은 보상 없이 버린다.

총 보상의 최댓값을 구하여라.

## 입력

첫째 줄에 N이 주어진다.
다음 N개의 줄에 마감 시한 D와 보상 W가 주어진다.

## 출력

받을 수 있는 보상의 최댓값을 출력한다.

## 예제 1

**입력**
```
4
2 50
1 10
2 30
1 40
```
**출력**
```
90
```

## 예제 2

**입력**
```
3
1 20
1 30
1 40
```
**출력**
```
40
```
