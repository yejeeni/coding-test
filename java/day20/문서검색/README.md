# Q3. 문서 검색

**유형**: 문자열 / 완전탐색
**제한**: 시간 2초 | 메모리 256MB | 문서 길이 ≤ 2500, 단어 길이 ≤ 50

## 문제

문서와 단어가 주어졌을 때, 단어가 문서에 겹치지 않고 최대 몇 번 등장하는지 구하라.

## 입력

첫째 줄에 문서, 둘째 줄에 단어가 주어진다.

## 출력

단어가 문서에 겹치지 않고 등장하는 최대 횟수를 출력한다.

## 예제 1

**입력**
```
ababababa
aba
```
**출력**
```
2
```

## 예제 2

**입력**
```
aaaa
aa
```
**출력**
```
2
```
