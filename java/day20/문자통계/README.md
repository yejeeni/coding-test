# Q1. 문자 통계

**유형**: 문자열 / charAt
**제한**: 시간 1초 | 메모리 256MB

## 문제

문자열이 한 줄 주어진다. 대문자, 소문자, 숫자, 공백의 개수를 각각 출력하라.

## 입력

공백을 포함한 문자열 한 줄.

## 출력

대문자 소문자 숫자 공백 순서로 공백으로 구분해 출력.

## 예제 1

**입력**
```
Hello World 123
```
**출력**
```
2 8 3 2
```

## 예제 2

**입력**
```
abc ABC 123
```
**출력**
```
3 3 3 2
```
