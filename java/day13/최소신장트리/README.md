## Q1. 최소 신장 트리 (워밍업)

**유형**: MST / 크루스칼

**문제**
N개의 도시가 있고, M개의 도로가 있다.
모든 도시를 연결하는 최소 비용을 구하라.

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 10,000
- 1 ≤ M ≤ 100,000
- 가중치: 1 ≤ w ≤ 1,000,000

입력 형식
첫 줄에 N, M
이후 M줄에 a b w (양방향, 가중치 w)

출력 형식
최소 신장 트리의 가중치 합

예제 입력
```
4 5
1 2 1
1 3 4
2 3 2
2 4 5
3 4 3
```

예제 출력
```
6
```
