## 재풀이 문제. 수업 듣기

**유형**: 그리디 / 정렬 (Comparator)

**문제**
N개의 수업이 있고, 각 수업은 시작 시간과 종료 시간이 주어진다.
한 번에 하나의 수업만 들을 수 있으며,
이전 수업 종료 시간과 다음 수업 시작 시간이 같아도 연속으로 들을 수 있다.
들을 수 있는 최대 수업 수를 구하라.

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000

입력 형식
첫 줄에 N
이후 N줄에 시작 시간, 종료 시간 (1 ≤ 시작 < 종료 ≤ 100,000)

출력 형식
들을 수 있는 최대 수업 수

예제 입력
```
5
1 3
2 3
3 5
3 6
5 7
```

예제 출력
```
3
```
