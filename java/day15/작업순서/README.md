## 작업 순서

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 작업이 있다. 일부 작업은 다른 작업이 완료된 후에만
시작할 수 있다. M개의 선후 관계가 주어질 때,
모든 작업을 완료할 수 있는 올바른 순서를 출력하여라.
단, 번호가 작은 작업을 우선으로 처리한다.
사이클이 있으면 -1을 출력한다.

### 제한
- 1 ≤ N ≤ 1,000
- 0 ≤ M ≤ 10,000

### 입력 형식
첫째 줄에 N, M이 주어진다.
다음 M개의 줄에 a b가 주어진다. (작업 a가 끝나야 작업 b 시작 가능)

### 출력 형식
올바른 작업 순서를 공백으로 구분하여 출력한다.
사이클이 있으면 -1을 출력한다.

### 예제 입력 1
```
5 4
1 2
1 3
2 4
3 4
```

### 예제 출력 1
```
1 2 3 4 5
```
