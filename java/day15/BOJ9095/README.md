## Q3. 1, 2, 3 더하기

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
정수 n을 1, 2, 3의 합으로 나타내는 방법의 수를 구하여라.
합을 나타낼 때는 수를 1개 이상 사용해야 한다.

### 제한
- 1 ≤ n ≤ 10
- 테스트 케이스 수: T

### 입력 형식
첫째 줄에 T가 주어진다.
각 테스트 케이스마다 정수 n이 주어진다.

### 출력 형식
각 테스트 케이스마다 n을 1, 2, 3의 합으로 나타내는 방법의 수를 출력한다.

### 예제 입력 1
```
3
4
7
10
```

### 예제 출력 1
```
7
44
274
```
