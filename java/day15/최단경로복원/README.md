## 최단경로 복원

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 도시와 M개의 단방향 도로가 있다.
도시 1에서 출발하여 도시 N까지 가는 최단경로를 구하되,
경로 자체(어떤 도시를 거치는지)도 함께 출력하여라.
최단경로가 여러 개면 그 중 하나만 출력한다.
도시 N에 도달할 수 없으면 -1을 출력한다.

### 제한
- 2 ≤ N ≤ 1,000
- 1 ≤ M ≤ 10,000
- 간선 가중치: 1 ≤ w ≤ 100

### 입력 형식
첫째 줄에 N, M이 주어진다.
다음 M개의 줄에 u v w가 주어진다.
(도시 u에서 v로 가는 가중치 w인 단방향 도로)

### 출력 형식
첫째 줄에 최단거리를 출력한다.
둘째 줄에 경로를 공백으로 구분하여 출력한다.
도달 불가능하면 -1을 출력한다.

### 예제 입력 1
```
5 7
1 2 2
1 3 3
2 3 1
2 4 5
3 4 1
3 5 8
4 5 2
```

### 예제 출력 1
```
6
1 3 4 5
```
