# Q2. 달팽이 배열
 
**유형**: 구현 / 방향 전환  
**제한**: 시간 1초 | N ≤ 100

## 문제
N×N 배열에 1부터 N²까지 달팽이 모양으로 채운 결과를 출력하라. 오른쪽→아래→왼쪽→위 순서로 나선형으로 채운다.

## 입력
N

## 출력
N×N 배열

## 예제 1
**입력**
```
3
```
**출력**
```
1 2 3
8 9 4
7 6 5
```

## 예제 2
**입력**
```
4
```
**출력**
```
1  2  3  4
12 13 14  5
11 16 15  6
10  9  8  7
```
