# Q1. 로봇 이동

**유형**: 구현 / 방향 벡터  
**제한**: 시간 1초 | N, M ≤ 100, K ≤ 1000

## 문제
N×M 격자에 로봇이 있다. 초기 위치 (r, c)에서 K개의 명령을 수행한다. 명령은 U(위), D(아래), L(왼쪽), R(오른쪽). 격자 밖으로 나가는 명령은 무시한다. 최종 위치를 출력하라.

## 입력
```
N M
r c
K
명령 문자열 (길이 K, 공백 없음)
```

## 출력
최종 위치 r c

## 예제 1
**입력**
```
5 5
3 3
7
ULURDDL
```
**출력**
```
3 2
```
