# 단어의 개수

**출처**: 백준 1152  
**유형**: 문자열 / split  
**제한**: 시간 2초 | 메모리 128MB

## 문제

영어 대소문자와 공백으로 이루어진 문자열이 주어진다. 이 문자열에 몇 개의 단어가 있는지 출력하라. 단, 공백이 연속해서 나올 수 있고 문자열의 앞뒤에도 공백이 있을 수 있다.

## 입력

한 줄에 영어 대소문자와 공백으로 이루어진 문자열 (길이 ≤ 1,000,000)

## 출력

단어의 개수

## 예제 1

**입력**
```
The first character is a blank
```
**출력**
```
6
```

## 예제 2

**입력**
```
 The first character is a blank
```
**출력**
```
6
```
