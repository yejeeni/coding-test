# 동전 거스름돈

**유형**: 무한 배낭 / 동전 DP
**제한**: 시간 1초 | 메모리 256MB | N ≤ 100 / M ≤ 10,000

## 문제

N가지 동전이 있다. 각 동전은 무한히 사용할 수 있다.
금액 M을 만드는 동전의 최소 개수를 출력하라.
만들 수 없으면 -1을 출력하라.

## 입력

첫째 줄에 N, M이 주어진다.
다음 줄에 N개의 동전 금액이 주어진다.

## 출력

최소 동전 개수 또는 -1

## 예제 1

**입력**
```
3 10
1 5 6
```
**출력**
```
2
```

## 예제 2

**입력**
```
2 3
2 4
```
**출력**
```
-1
```
