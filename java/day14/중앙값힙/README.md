## 중앙값 힙

**유형**: PriorityQueue / Comparator

**문제**
정수가 하나씩 들어올 때마다 지금까지 입력된 수의 중앙값을 출력하라.
수의 개수가 짝수일 때는 중앙값 중 작은 것을 출력한다.

**제한**
- 시간: 0.1초 / 메모리: 256MB
- 입력 횟수 N: 1 ≤ N ≤ 100,000
- 입력값: -10,000 ≤ x ≤ 10,000

입력 형식
첫 줄에 N
이후 N줄에 정수 하나씩

출력 형식
각 정수가 입력될 때마다 중앙값을 한 줄씩 출력

예제 입력
```
7
1
5
2
10
-99
7
5
```

예제 출력
```
1
1
2
2
2
2
5
```
