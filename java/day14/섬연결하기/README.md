## 문제 2. 섬 연결하기

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 섬을 다리로 모두 연결하려 한다.
M개의 다리 후보가 주어질 때, 모든 섬을 연결하는 데 필요한
다리 건설 비용의 최솟값을 구하여라.
모든 섬을 연결할 수 없으면 -1을 출력한다.

### 제한
- 1 ≤ N ≤ 1,000
- 1 ≤ M ≤ 10,000
- 다리 비용: 1 ≤ w ≤ 1,000,000
- 두 섬 사이에 다리 후보가 여러 개일 수 있음
- 무방향 그래프

### 입력 형식
첫째 줄에 N, M이 주어진다.
다음 M개의 줄에 a b w가 주어진다. (섬 a와 b를 잇는 비용 w인 다리 후보)

### 출력 형식
최소 비용을 출력한다. 연결 불가능하면 -1을 출력한다.

### 예제 입력 1
```
4 5
1 2 3
1 3 1
2 3 2
2 4 5
3 4 4
```

### 예제 출력 1
```
7
```
