## 합이 S인 부분집합

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 자연수로 이루어진 배열이 있다. 이 배열에서 원소를 골라 합이 정확히 S가 되는 부분집합의 수를 구하라.

### 제한
- 1 ≤ N ≤ 20
- 1 ≤ 배열의 원소 ≤ 100
- 1 ≤ S ≤ 2000

### 입력 형식
첫째 줄에 N과 S가 공백으로 주어진다.
둘째 줄에 N개의 정수가 공백으로 주어진다.

### 출력 형식
합이 S인 부분집합의 수를 출력한다.
(빈 부분집합은 제외한다. 단, S = 0이면 1로 출력한다.)

### 예제 입력 1
```
5 10
1 2 3 4 5
```

### 예제 출력 1
```
3
```
