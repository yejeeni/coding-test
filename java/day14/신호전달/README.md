## 문제 1. 신호 전달

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 서버(1~N번)와 M개의 단방향 네트워크 연결이 있다.
서버 K에서 신호를 보낼 때, 모든 서버에 신호가 도달하기까지
걸리는 최소 시간을 구하여라.
단, 신호가 도달하지 못하는 서버가 하나라도 있으면 -1을 출력한다.

### 제한
- 1 ≤ N ≤ 100
- 1 ≤ M ≤ 6,000
- 1 ≤ K ≤ N
- 간선 가중치: 1 ≤ w ≤ 100
- 자기 자신으로 가는 간선 없음

### 입력 형식
첫째 줄에 N, M, K가 공백으로 주어진다.
둘째 줄부터 M개의 줄에 u v w가 주어진다. (서버 u에서 v로 가는 가중치 w인 단방향 간선)

### 출력 형식
모든 서버에 신호가 도달하는 최소 시간을 출력한다.
도달하지 못하는 서버가 있으면 -1을 출력한다.

### 예제 입력 1
```
4 5 1
1 2 1
1 3 4
2 3 2
2 4 5
3 4 1
```

### 예제 출력 1
```
4
```
