# 정수 삼각형

**출처**: 백준 1932
**유형**: 2D DP
**제한**: 시간 1초 | 메모리 128MB | N ≤ 500

## 문제

```
        7
      3   8
    8   1   0
  2   7   4   4
4   5   2   6   5
```

위 삼각형 꼭대기부터 아래쪽으로 내려오면서 숫자를 선택해요. 아래 줄로 내려갈 때 **왼쪽 자식** 또는 **오른쪽 자식** 숫자만 선택 가능해요. 선택한 수의 합의 최댓값을 구하세요.

## 입력

첫째 줄: N (삼각형 행 수)
이후 N줄: 각 행의 숫자

## 출력

합의 최댓값

## 예제 1

**입력**
```
5
7
3 8
8 1 0
2 7 4 4
4 5 2 6 5
```
**출력**
```
30
```
