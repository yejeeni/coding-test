## 문제 2. 사이클 감지 [메인]

시간 제한: 1초 | 메모리 제한: 256MB | N ≤ 10,000

N개의 노드, M개의 간선으로 이루어진 무방향 그래프가 있다.
간선을 순서대로 하나씩 추가할 때,
해당 간선을 추가하면 사이클이 생기면 "YES", 아니면 "NO"를 출력하라.

입력 형식
첫째 줄에 N, M이 주어진다.
이후 M줄에 걸쳐 간선 a b가 주어진다.

출력 형식
M줄에 걸쳐 YES 또는 NO를 출력한다.

예제 입력
```
4 5
1 2
1 3
2 3
3 4
2 4
```

예제 출력
```
NO
NO
YES
NO
YES
```
