## 문제: 괄호 유효성 검사

[백준 스타일 / 실버 5]
시간 제한: 1초 | 메모리 제한: 256MB

여러 줄의 괄호 문자열이 주어진다.
각 문자열이 올바른 괄호 문자열인지 판단하라.

올바른 괄호 문자열의 조건:
- '('와 ')'만 포함한다.
- 모든 '('는 대응하는 ')'가 있다.
- ')'가 '('보다 먼저 나올 수 없다.

입력 형식
첫째 줄에 테스트케이스 수 T가 주어진다.
이후 T줄에 걸쳐 괄호 문자열이 하나씩 주어진다.
문자열 길이 ≤ 100

출력 형식
각 줄에 "YES" 또는 "NO"를 출력한다.

예제 입력
```
3
(())()
)(
(()
```

예제 출력
```
YES
NO
NO
```
