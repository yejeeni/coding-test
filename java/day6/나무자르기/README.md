# 나무 자르기

## 문제
벌목꾼이 절단기를 이용해 나무를 자르려 한다.
절단기 높이를 H로 설정하면, 높이가 H보다 큰 나무는 H 위 부분이 잘려 나온다.
잘려 나온 나무의 합이 정확히 M미터 이상이 되어야 한다.
절단기 높이 H의 최댓값을 구하라. (H는 0 이상의 정수)

## 제한
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 1,000,000
- 1 ≤ M ≤ 2,000,000,000
- 1 ≤ 나무 높이 ≤ 1,000,000,000
- 항상 M 이상의 나무를 가져갈 수 있는 입력만 주어진다

## 입력 형식
```
첫째 줄: N M (나무의 수, 필요한 나무 길이)
둘째 줄: N개의 나무 높이 (공백 구분)
```

## 출력 형식
```
절단기 높이 H의 최댓값 출력
```

## 예제 입력 1
```
4 7
20 15 10 17
```

## 예제 출력 1
```
15
```

## 예제 입력 2
```
5 10
1 2 3 4 5
```

## 예제 출력 2
```
1
```
