### 🟢 문제 1 — 이진탐색 직접 구현 (워밍업)

**[백준 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 배열은 오름차순 정렬되어 주어짐
- 1 ≤ 각 수 ≤ 1,000,000

**문제**
정렬된 N개의 수와 Q개의 쿼리가 주어진다.
각 쿼리마다 해당 수가 배열에 존재하면 인덱스(0부터 시작)를,
존재하지 않으면 -1을 출력하라.
단, Arrays.binarySearch() 사용 금지.

**입력 형식**
```
첫째 줄: N
둘째 줄: N개의 정수 (오름차순)
셋째 줄: Q
다음 Q줄: 쿼리 수 하나씩
```

**출력 형식**
```
Q개의 결과를 한 줄에 하나씩
```

**예제 입력 1**
```
5
1 3 5 7 9
3
1
6
9
```

**예제 출력 1**
```
0
-1
4
```

💡 체크포인트: while (lo <= hi) 루프, mid = (lo + hi) / 2, 찾으면 mid 반환, 못 찾으면 -1.
