# 격자 경로 수

**유형**: 2D DP / 격자 경로
**제한**: 시간 1초 | 메모리 256MB | N, M ≤ 100

## 문제

N×M 격자의 (1,1)에서 (N,M)까지 오른쪽 또는 아래로만 이동합니다. 격자 일부 칸에 장애물(1)이 있어 지나갈 수 없습니다. 이동 가능한 경로의 수를 출력하세요.

## 입력

첫째 줄에 N, M. 다음 N개의 줄에 격자 정보 (0: 빈칸, 1: 장애물)

## 출력

(1,1)에서 (N,M)까지의 경로 수.

## 예제 1

**입력**
```
3 3
0 0 0
0 1 0
0 0 0
```
**출력**
```
2
```

## 예제 2

**입력**
```
2 3
0 0 1
0 0 0
```
**출력**
```
2
```
