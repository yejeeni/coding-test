
## 🔥 문제 3 (도전) — 행렬 안장점
`[백준 스타일]`

N×M 크기의 정수 행렬이 주어진다.

다음 두 값을 구하시오:
*   **A** = 각 행의 최솟값들 중 최댓값 (row-min의 max)
*   **B** = 각 열의 최댓값들 중 최솟값 (col-max의 min)

A와 B가 같으면 `SADDLE POINT: {값}` 을 출력하고,
다르면 `A={A값} B={B값}` 을 출력하시오.

**📌 제한**
*   **시간:** 1초 / **메모리:** 256MB
*   1 ≤ N, M ≤ 1,000
*   각 원소: -10^9 ≤ x ≤ 10^9

**예제 입력 1**
```text
3 3
1 2 3
4 5 6
7 8 9
```

**예제 출력 1**
```text
SADDLE POINT: 7
```

**예제 입력 2**
```text
2 3
3 1 4
1 5 9
```

**예제 출력 2**
```text
A=1 B=3
```
