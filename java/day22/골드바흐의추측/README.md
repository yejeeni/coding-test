# 골드바흐의 추측

**출처**: 백준 9020  
**유형**: 수학 / 에라토스테네스의 체  
**제한**: 시간 1초 | 메모리 256MB | n ≤ 10,000

## 문제

1보다 큰 자연수 중 1과 자기 자신만을 약수로 가지는 수를 소수라 한다.

골드바흐의 추측: **2보다 큰 모든 짝수는 두 소수의 합으로 나타낼 수 있다.**

짝수 n이 주어졌을 때, n을 두 소수 p, q의 합(p ≤ q)으로 나타내되 **p와 q의 차이가 가장 작은 것**을 출력하라.

## 입력

첫째 줄: 테스트 케이스의 수 T  
각 줄: 4 이상 10000 이하의 짝수 n

## 출력

각 테스트 케이스에 대해 p와 q를 공백으로 구분해 출력 (p ≤ q)

## 예제 1

**입력**
```
3
8
12
16
```
**출력**
```
3 5
5 7
5 11
```
