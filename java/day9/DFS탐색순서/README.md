# Day9-Q1. DFS 탐색 순서

[출처 스타일: 백준]

[시간 제한: 1초 / 메모리 제한: 256MB]

[N ≤ 1,000 / M ≤ 10,000]

무방향 그래프가 주어질 때, 1번 노드에서 시작하는 DFS 탐색 순서를 출력하라.
인접한 노드가 여러 개일 때는 번호가 작은 노드부터 방문한다.
그래프는 연결 그래프(모든 노드가 연결됨)임을 보장한다.

입력 형식
```
첫째 줄: N M (노드 수, 간선 수)
다음 M줄: 간선의 양 끝 노드 A B
```

출력 형식
```
DFS 방문 순서를 공백으로 구분해 한 줄에 출력
```

예제 입력 1
```
4 4
1 2
1 3
2 4
3 4
```

예제 출력 1
```
1 2 4 3
```

예제 입력 2
```
5 5
1 2
1 3
2 4
3 4
4 5
```

예제 출력 2
```
1 2 4 3 5
```
