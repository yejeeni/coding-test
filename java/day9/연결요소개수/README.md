# Day9-Q2. 연결 요소 개수

[출처 스타일: 백준]

[시간 제한: 1초 / 메모리 제한: 256MB]

[N ≤ 1,000 / M ≤ 10,000]

무방향 그래프에서 연결 요소(Connected Component)의 개수를 구하라.
연결 요소란 서로 연결된 노드들의 최대 집합이다.

입력 형식
```
첫째 줄: N M (노드 수, 간선 수)
다음 M줄: 간선의 양 끝 노드 A B
```

출력 형식
```
연결 요소의 개수를 출력
```

예제 입력 1
```
6 4
1 2
2 3
4 5
5 6
```

예제 출력 1
```
2
```

예제 입력 2
```
5 0
```

예제 출력 2
```
5
```
