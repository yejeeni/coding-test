# Day9-Q3. 사이클 감지

[출처 스타일: 백준]
[시간 제한: 1초 / 메모리 제한: 256MB]
[N ≤ 1,000 / M ≤ 10,000]

무방향 그래프에서 사이클(cycle)이 존재하면 YES, 없으면 NO를 출력하라.
사이클이란 한 노드에서 출발해 자기 자신으로 돌아오는 경로가 존재하는 것이다.

입력 형식
```
첫째 줄: N M (노드 수, 간선 수)
다음 M줄: 간선의 양 끝 노드 A B
```

출력 형식
```
YES 또는 NO
```

예제 입력 1
```
4 4
1 2
2 3
3 4
4 2
```

예제 출력 1
```
YES
```

예제 입력 2
```
5 4
1 2
2 3
3 4
4 5
```

예제 출력 2
```
NO
```
