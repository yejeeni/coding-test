

### 🔴 문제 3 — 두 스택으로 큐 만들기 (도전)

**[프로그래머스 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ 명령 수 ≤ 100,000
- push 값: 1 ≤ v ≤ 10,000
- `int[]` 기반 스택 2개만 사용할 것 (`ArrayDeque` 금지)

**문제**
스택 두 개(`inStack`, `outStack`)만을 이용해서 큐의 동작을 구현하라.

명령은 세 종류:
- `push v` — 값 v를 큐에 삽입
- `pop` — 큐의 맨 앞 값을 제거하고 출력. 큐가 비어있으면 `-1` 출력
- `size` — 현재 큐에 들어있는 원소 수 출력

**입력 형식**
```
첫째 줄: 명령 수 M
다음 M줄: 명령
```

**출력 형식**
```
pop, size 명령마다 한 줄씩 출력
```

**예제 입력 1**
```
7
push 1
push 2
push 3
pop
size
pop
pop
```
**예제 출력 1**
```
1
2
2
3
```

**예제 입력 2**
```
4
pop
push 5
size
pop
```
**예제 출력 2**
```
-1
1
5
```

> 💡 핵심 아이디어: `pop`할 때 `outStack`이 비어있으면 `inStack`의 원소를 전부 `outStack`으로 옮겨요. 왜 이게 큐가 되는지 생각하면서 구현해보세요.
