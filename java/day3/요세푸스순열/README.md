### 🟡 문제 2 — 요세푸스 순열 (메인)

**[백준 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 1,000
- 1 ≤ K ≤ N

**문제**
1번부터 N번까지 N명이 원형으로 앉아 있다. 1번부터 시작해서 K번째 사람을 제거하고, 다음 사람부터 다시 세기를 반복한다. 제거되는 순서를 출력하라.

단, **`ArrayDeque`를 직접 구현한 원형 큐**로 풀 것. (배열 크기 N+1의 원형 큐, front/rear 인덱스 방식)

**입력 형식**
```
한 줄: N K
```

**출력 형식**
```
<N1, N2, N3, ...> 형태로 한 줄 출력
```

**예제 입력 1**
```
5 2
```
**예제 출력 1**
```
<2, 4, 1, 5, 3>
```

**예제 입력 2**
```
7 3
```
**예제 출력 2**
```
<3, 6, 2, 7, 5, 1, 4>
```

> 💡 원형 큐 핵심: `rear = (rear + 1) % capacity` 로 인덱스를 순환시켜요. enqueue / dequeue / isEmpty 세 메서드만 만들면 돼요.
