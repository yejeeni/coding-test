### 🟡 문제 2 — 단어 빈도수 1위 (메인)

**[프로그래머스 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 각 단어 길이 ≤ 20, 알파벳 소문자로만 구성
- 빈도수 1위 단어는 항상 유일하게 존재한다

**문제**
N개의 단어가 한 줄에 하나씩 주어진다. 가장 많이 등장한 단어와 그 등장 횟수를 출력하라.

**입력 형식**
```
첫째 줄: N
다음 N줄: 단어 하나씩
```

**출력 형식**
```
단어 횟수
```

**예제 입력 1**
```
7
apple
banana
apple
orange
banana
apple
grape
```
**예제 출력 1**
```
apple 3
```

**예제 입력 2**
```
4
java
java
python
java
```
**예제 출력 2**
```
java 3
```
