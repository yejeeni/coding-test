### 🟢 문제 1 — 정수 배열 합계와 최댓값 (워밍업)

**[백준 스타일]**

**제한**
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- −10⁹ ≤ 각 정수 ≤ 10⁹

**문제**
N개의 정수가 주어진다. 모든 수의 합과 최댓값을 출력하라.

**입력 형식**
```
첫째 줄: N
둘째 줄: N개의 정수 (공백 구분)
```

**출력 형식**
```
합 최댓값
(한 줄, 공백 구분)
```

**예제 입력 1**
```
5
3 -1 4 1 5
```
**예제 출력 1**
```
12 5
```

**예제 입력 2**
```
3
-5 -3 -9
```
**예제 출력 2**
```
-17 -3
```
