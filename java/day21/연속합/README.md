# 1번. 연속합 (백준 1912)

**유형**: 1D DP
**제한**: 시간 2초 | 메모리 128MB | N ≤ 100,000

## 문제

N개의 정수로 이루어진 수열이 있다. 연속된 몇 개의 수를 선택해서 구할 수 있는 합 중 가장 큰 합을 구하라. 단, 적어도 하나는 선택해야 한다.

## 입력

첫째 줄에 N, 둘째 줄에 N개의 정수.

## 출력

최대 연속합을 출력한다.

## 예제 1
**입력**
```
10
2 1 -4 3 4 -4 6 5 -5 1
```
**출력**
```
14
```

## 예제 2
**입력**
```
5
-1 -2 -3 -4 -5
```
**출력**
```
-1
```
