# 2번. 잃어버린 괄호 (백준 1541)

**유형**: 그리디 / 문자열
**제한**: 시간 2초 | 메모리 128MB

## 문제

세준이는 양수와 `+`, `-`로만 이루어진 수식을 적었다. 여기에 괄호를 적절히 추가해 결과를 최소로 만들려 한다. 괄호는 마음대로 추가할 수 있다.

## 입력

수식. 길이 ≤ 50, 피연산자는 0~9로 이루어진 수 (선행 0 가능).

## 출력

최솟값 출력.

## 예제 1
**입력**
```
55-50+40
```
**출력**
```
-35
```

## 예제 2
**입력**
```
10+20+30
```
**출력**
```
60
```
