# [메인] 단어 빈도수

## 문제
N개의 단어가 주어진다. 각 단어의 등장 횟수를 구하고,
빈도수 내림차순으로 출력하라. 빈도수가 같으면 알파벳 오름차순으로 출력한다.

## 제한
- 시간: 1초 / 메모리: 256MB
- 1 ≤ N ≤ 100,000
- 단어는 소문자 알파벳만으로 이루어짐
- 단어 길이 ≤ 20

## 입력 형식
```
첫째 줄: N
다음 N줄: 단어
```

## 출력 형식
```
"단어 횟수" 형식으로 한 줄씩 출력
```

## 예제 입력 1
```
7
apple
banana
apple
cherry
banana
apple
cherry
```

## 예제 출력 1
```
apple 3
banana 2
cherry 2
```

## 예제 입력 2
```
3
a
b
a
```

## 예제 출력 2
```
a 2
b 1
```
