# [워밍업] 절댓값 힙

## 문제
정수를 삽입하거나, 절댓값이 가장 작은 수를 출력·제거하는 연산을 처리하라.
절댓값이 같으면 음수를 먼저 출력한다. 힙이 비어있을 때 출력 연산이 들어오면 0을 출력한다.

## 제한
- 시간: 1초 / 메모리: 256MB
- 1 ≤ 연산 수 N ≤ 100,000
- 삽입하는 정수: -10^9 이상 10^9 이하, 0은 삽입되지 않음

## 입력 형식
```
첫째 줄: N (연산 수)
다음 N줄: 정수 x
x ≠ 0이면 힙에 x 삽입
x == 0이면 절댓값 최솟값 출력·제거
```

## 출력 형식
```
출력 연산마다 한 줄씩 출력
```

## 예제 입력 1
```
5
1
-1
0
-2
0
```

## 예제 출력 1
```
-1
1
```

## 예제 입력 2
```
3
0
0
0
```

## 예제 출력 2
```
0
0
0
```
