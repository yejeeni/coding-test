## 문제: 빈도수 TOP K

시간 제한: 1초 | 메모리 제한: 256MB

문자열 배열 words가 주어진다.
각 단어의 등장 횟수를 세고,
등장 횟수가 높은 순서대로 K개의 단어를 출력하라.

등장 횟수가 같으면 알파벳 오름차순으로 정렬한다.

입력 형식
```
첫째 줄에 단어의 수 N과 K가 주어진다. (1 ≤ K ≤ N ≤ 100,000)
이후 N줄에 걸쳐 단어가 하나씩 주어진다. (단어 길이 ≤ 20, 소문자만)
```

출력 형식
```
등장 횟수 내림차순, 횟수 같으면 알파벳 오름차순으로 K개 출력한다.
```

예제 입력
```
9 3
apple
banana
apple
cherry
banana
apple
cherry
banana
cherry
```

예제 출력
```
apple 3
banana 3
cherry 3
```
