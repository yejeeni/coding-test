## Q2. 특정 거리의 도시 찾기 (메인)

**유형**: 다익스트라 응용  

**문제**  
N개의 도시, M개의 단방향 도로가 있다. 도로의 가중치는 모두 1이다.  
출발 도시 K에서 최단거리가 **정확히 X**인 도시를 모두 구해라.

**제한**
- 시간: 2초 / 메모리: 512MB
- 1 ≤ N ≤ 300,000
- 1 ≤ M ≤ 1,000,000
- 가중치 = 1 (균일)
- 해당 도시가 없으면 -1 출력

입력 형식  
첫 줄에 N M K X
이후 M줄: `u v` — u→v 방향 도로 (가중치는 모두 1이라 명시 안 함)

출력 형식  
최단거리가 X인 도시 번호를 오름차순으로 한 줄씩 출력. 없으면 -1

예제 입력
```
4 3 1 2
1 2
2 3
2 4
```

예제 출력
```
3
4
```
