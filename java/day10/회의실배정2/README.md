## 회의실 배정 II

**유형**: PriorityQueue / Greedy  
**시간 제한**: 1초 / **메모리**: 256MB / **N ≤ 100,000**

회의가 N개 있다. 각 회의는 시작 시간과 끝 시간이 주어진다.  
회의는 끝 시간이 되면 회의실을 즉시 반납한다.  
(끝 시간 = 다음 회의 시작 시간이면 같은 방 사용 가능)

회의를 모두 진행하기 위해 필요한 **최소 회의실 수**를 구하여라.

**입력 형식**  
```
첫째 줄: N (1 ≤ N ≤ 100,000)  
둘째 줄부터 N줄: 시작 시간 Si, 끝 시간 Ei (1 ≤ Si < Ei ≤ 10^9)
```

**출력 형식**
```
필요한 최소 회의실 수
```

예제 입력 1
```
5  
1 5  
2 3  
3 7  
4 6  
6 8
```

예제 출력 1
```
3
```

예제 입력 2
```
3  
1 3  
2 5  
3 6
```

예제 출력 2
```
2
```
