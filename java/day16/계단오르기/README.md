## Q1. 계단 오르기

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 계단이 있고 각 계단에 점수가 있다.
아래 규칙을 지키며 계단 꼭대기(N번째)까지 올라갈 때
얻을 수 있는 최대 점수를 구하여라.

규칙:
1. 한 번에 1칸 또는 2칸 오를 수 있다.
2. 연속으로 3칸을 연속해서 밟으면 안 된다.
3. 마지막 계단(N번째)은 반드시 밟아야 한다.

### 제한
- 1 ≤ N ≤ 300
- 각 계단 점수: 1 ≤ score ≤ 10,000

### 입력 형식
첫째 줄에 N이 주어진다.
둘째 줄에 N개의 계단 점수가 주어진다.

### 출력 형식
얻을 수 있는 최대 점수를 출력한다.

### 예제 입력 1
```
6
10 20 15 25 10 20
```

### 예제 출력 1
```
75
```
