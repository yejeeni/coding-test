## 비상 탈출

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 건물과 M개의 단방향 도로가 있다.
재난이 발생해 건물 1에서 출발하여 대피소(건물 N)까지
이동하려 한다. 반드시 중간 기착지 K를 경유해야 한다.
1 → K → N 경로의 최단 거리를 구하여라.
경로가 존재하지 않으면 -1을 출력한다.

### 제한
- 2 ≤ N ≤ 1,000
- 1 ≤ M ≤ 10,000
- 1 ≤ K ≤ N
- 간선 가중치: 1 ≤ w ≤ 100

### 입력 형식
첫째 줄에 N, M, K가 주어진다.
다음 M개의 줄에 u v w가 주어진다.
(건물 u에서 v로 가는 가중치 w인 단방향 도로)

### 출력 형식
최단 거리를 출력한다. 경로가 없으면 -1을 출력한다.

### 예제 입력 1
```
5 7 3
1 2 2
1 3 3
2 3 1
2 4 5
3 4 1
3 5 8
4 5 2
```

### 예제 출력 1
```
6
```
