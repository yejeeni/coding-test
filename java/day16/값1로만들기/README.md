## Q2. 1로 만들기

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
정수 N이 주어질 때, 아래 세 가지 연산을 적절히 사용하여
1로 만드는 데 필요한 최소 연산 횟수를 구하여라.

1. N이 3으로 나누어 떨어지면 3으로 나눈다.
2. N이 2로 나누어 떨어지면 2로 나눈다.
3. N에서 1을 뺀다.

### 제한
- 1 ≤ N ≤ 1,000,000

### 입력 형식
첫째 줄에 N이 주어진다.

### 출력 형식
최소 연산 횟수를 출력한다.

### 예제 입력 1
```
10
```

### 예제 출력 1
```
3
```
