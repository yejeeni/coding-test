## 네트워크 최소 비용

**시간 제한**: 1초 | **메모리 제한**: 256MB

### 문제
N개의 컴퓨터를 모두 하나의 네트워크로 연결하려 한다.
M개의 케이블 후보가 주어질 때, 모든 컴퓨터를 연결하는
최소 비용을 구하여라. 연결이 불가능하면 -1을 출력한다.

### 제한
- 1 ≤ N ≤ 1,000
- 1 ≤ M ≤ 10,000
- 케이블 비용: 1 ≤ w ≤ 1,000,000
- 무방향 그래프

### 입력 형식
첫째 줄에 N, M이 주어진다.
다음 M개의 줄에 a b w가 주어진다.

### 출력 형식
최소 비용을 출력한다. 불가능하면 -1을 출력한다.

### 예제 입력 1
```
4 5
1 2 3
1 3 1
2 3 2
2 4 5
3 4 4
```

### 예제 출력 1
```
7
```
