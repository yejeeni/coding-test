팀 나누기

시간 제한: 1초 | 메모리: 256MB | N ≤ 100,000, M ≤ 100,000

문제:

N명의 선수와 M개의 정보가 주어진다.
각 정보는 "선수 a와 선수 b는 같은 팀이다"를 나타낸다.
모든 정보를 처리한 뒤, 총 몇 개의 팀이 존재하는지 출력하라.

입력 형식:

첫째 줄에 N, M이 주어진다.
다음 M줄에 a, b가 주어진다. (1 ≤ a, b ≤ N)

출력 형식:

팀의 수를 출력한다.

예제 입력 1:
```
6 4
1 2
2 3
4 5
5 6
```

예제 출력 1:
```
2
```

예제 입력 2:
```
5 0
```

예제 출력 2:
```
5
```
