# 반시계방향 회전

**유형**: 구현 / 2D 배열 회전  
**제한**: 시간 1초 | 메모리 256MB | 1 ≤ N, M ≤ 100, 1 ≤ K ≤ 4

## 문제

N×M 크기의 2차원 배열이 주어진다. 이 배열을 반시계방향으로 90도씩 K번 회전한 결과를 출력하시오.

## 입력

첫째 줄에 N M K가 주어진다.  
둘째 줄부터 N개의 줄에 배열의 원소가 주어진다.

## 출력

K번 회전한 배열을 출력한다.

## 예제 1
**입력**
```
2 3 1
1 2 3
4 5 6
```
**출력**
```
3 6
2 5
1 4
```

## 예제 2
**입력**
```
3 3 2
1 2 3
4 5 6
7 8 9
```
**출력**
```
9 8 7
6 5 4
3 2 1
```
